# virtualenv

# Install
```
pip install virtualenv
```

# Create Virtual
```
virtualenv project1_env
```

# Activate Folder
```
source  project1_env/bin/activate
```

# Listar Libreria de PIP
```
pip list
```

# Salir del ambiente
```
deactivate
```
# Librerias
```
pip freeze --local > requeriments.txt
```
